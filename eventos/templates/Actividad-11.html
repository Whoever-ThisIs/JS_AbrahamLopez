Eventos y mas eventos

Nombre de evento Descripción Ejemplo
.onfocus y .onblur Estos eventos suceden cuando de hace focus sobre un elemento del documento y cuando se quita el focus del elemento. Intenta llenarme:
.ondblclick Este evento se da cuando se le da doble click al elemento. Dame doble click
.onkeypress El evento abarca cuando se presiona una tecla y se deja de presionar. Pulsa cualquier tecla para aumentar el numero
.onkeydown El evento abarca cuando se presiona una tecla (no incluye cuando se suelta) Dale focus al input y cambia el color de este input usando las flechas
.onmouseover El evento se activa cuando pones el mouse encima del elemento quieres ponerte encima del input?
.onmouseover El evento se activa cuando quitas el mouse del elemento
.onmousemove El evento se activa pones el mouse encima del elemento Soy adivino y puedo saber cuando mueves el mouse
.onmousedown El evento se activa pones el mouse encima del elemento Soy adivino y puedo saber cuando haces click en cualquier parte de la pantalla
.onmouseup El evento se activa cuando quitas el mouse del elemento
.onresize El evento se activa cuando has cambiado el tamaño de la ventana. Soy adivino y puedo saber cuando haces click en cualquier parte de la pantalla
.onscroll El evento se activa cuando seleccionas un texto. Soy adivino y puedo saber cuando haces scroll en la pantalla
